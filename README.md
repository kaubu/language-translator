# language-translator
Lookup words for many Ancient Languages.

## Built-in languages
* Ancient Greek
* Latin
* Old Norse
* Old English
* Middle English
* Old French
* Middle French
* French (Modern)
* Old High German
* Old Irish
* Middle Irish

## Notes
### Latin
Most Latin verbs start with "I" as in "I like". Some Latin verbs start with "to" like "to hate".
